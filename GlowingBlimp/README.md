# Glowing Blimp (Custom Mushroom)

*Glowing Blimp* is a custom mushroom added using the MushroomPatcher mod. It serves as a practical example of how to use the Custom Mushroom Template to add mushrooms to the game.

It is available on the [Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3679319195).
